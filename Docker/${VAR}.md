```js
${VAR} -> значение VAR

${VAR:-default} -> значение VAR если задано и непустое, в противном случае default
${VAR-default} -> значение VAR если установлено, в противном случае default

${VAR:?error} -> значение VAR, если оно задано и непустое, в противном случае завершается с ошибкой
${VAR?error} -> значение VAR если установлено, в противном случае завершите работу с ошибкой

${VAR:+replacement} -> значение replacement если VAR установлено и непустое, в противном случае пустое
${VAR+replacement} -> значение replacement если VAR установлено, в противном случае пусто
```
